🎧 Screen Sound

Aplicação de console desenvolvida em C# para cadastro, avaliação e análise de bandas.

📌 Sobre o projeto

O Screen Sound é um projeto simples focado em praticar conceitos fundamentais da linguagem C#.
A aplicação permite registrar bandas, atribuir notas e visualizar a média das avaliações.

⚙️ Funcionalidades
➕ Registrar novas bandas
📃 Listar bandas cadastradas
⭐ Avaliar uma banda
📊 Exibir média de avaliações por banda
🔁 Menu interativo via console

🧠 Conceitos aplicados
Estrutura básica de um programa em C#
Tipos de dados e variáveis
Funções (métodos)
Listas (List<T>)
Dicionários (Dictionary<TKey, TValue>)
Estruturas de repetição
Condicionais (if/else e switch)
Manipulação de dados
Interação com usuário via Console

🗂️ Estrutura do projeto
ExibirLogo() → Mostra o banner inicial
ExibirOpcoesMenu() → Controla o fluxo do sistema
RegistrarBanda() → Adiciona novas bandas
MostrarBandasRegistradas() → Lista bandas cadastradas
AvaliarUmaBanda() → Permite adicionar notas
ExibirMedia() → Calcula média das avaliações

▶️ Como executar
Clone o repositório
Abra o projeto em uma IDE (Visual Studio, VS Code, etc.)
Execute o arquivo principal
